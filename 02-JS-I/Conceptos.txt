Las variables son simbolos o letras a las que se les puede asignar un valor y que posteriomente pueden ser manipuladas de acuerdo a las necesidades del programa. Las variables pueden ser de varios tipos en función del valor que contienen. Pueden ser números ó cadenas de carácteres (strings) o de tipo boolean (falso/verdadero, 0/1, etc). Las funciones son "paquetes de instrucciones" a las que se les pone un nombre que define que es lo que hacen y si durante la compilación del programa se cumplen las condiciones, manipulan los valores de entrada para obtener un resultado que puede ser impreso en pantalla con el comando return, la misma función puede imprimir con el mismo comando una salida cuando las condiciones no se cumplen para entrar a la función. Los argumentos son la forma de enriquecer una función generalmente son expresiones. Las declaraciones son palabras reservadas por el compilador que indican una instrucción que el programa puede realizar con los argumentos que nosotros proporcionamos, en el caso de if nos sirve para controlar el flujo de ejecución si se cumple o no una condición. Los valores booleanos son un tipo de datos en donde solo tenemos 2 opciones (verdadero y falso).
